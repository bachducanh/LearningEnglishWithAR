Hướng dẫn sử dụng sản phẩm sau khi đã phát triển:
Bước 1:

Tải folder có tên “Sản phẩm sau khi đã phát triển thêm”, sau đó mở Visual Studio Code (VS Code) và chọn Open Folder, sau đó truy cập vào thư mục “Sản phẩm sau khi đã phát triển thêm”.

Bước 2:

Khởi tạo môi trường ảo (virtual environment) và cài đặt các thư viện cần thiết bằng cách mở Terminal mới trong VS Code, sau đó nhập lệnh sau:

pip install ultralytics opencv-python cvzone numpy torch torchvision pillow filterpy tqdm pygame gtts


Bước 3:

Sau khi quá trình cài đặt hoàn tất, mở file "Sản phẩm phát triển thêm.py" và khởi động chương trình để trải nghiệm mini game.
